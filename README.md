.tfstate - игнорируются все файлы, в имени которых присутствует .tfstate  
.tfvars - игнорируются все файлы, в имени которых присутствует .tfvars  
_override.tf - игнорируются все файлы, в имени которых есть _override.tf  
crash.*.log - игнорируются все файлы, начинающиеся с crash., заканчивающиеся на .log и имеющие любой текст между ними  
tfplan (закомментировано) - пример правила для игнорирования файлов, содержащих tfplan в имени  
.terraform/ - игнорируется вся директория .terraform и её содержимое  
crash.log, .terraformrc, terraform.rc, .terraform.tfstate.lock.info - игнорируются файлы с указанными именами
