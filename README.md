# \# Игнорируемые файлы

# 

# \## В terraform/.gitignore:

# \- Локальные директории `.terraform/` с плагинами и кэшем

# \- Файлы состояния Terraform (`.tfstate`, `.tfstate.\*`), которые могут содержать чувствительные данные

# \- Файлы логов аварийных завершений (`crash.log`)

# \- Автоматически сгенерированные файлы переменных (`.tfvars`, `.tfvars.json`)

# \- Файлы переопределений (`override.tf`, `\*\_override.tf`)

# \- Файлы конфигурации CLI (`.terraformrc`, `terraform.rc`)

#
